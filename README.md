# Blinkit-Data-Analysis-
# 🛒 Blinkit Data Analysis using Python

## 📌 Project Overview

This project focuses on analyzing Blinkit grocery sales data to uncover meaningful insights about sales performance, customer preferences, and outlet trends. The analysis helps in understanding how different factors like outlet type, item category, and fat content impact total sales.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Analyze sales trends across different outlet types
* Compare performance of item categories (Low Fat vs Regular)
* Generate visual insights using charts
* Support data-driven decision making

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib

---

## 📂 Dataset

The dataset contains information about:

* Outlet Location Type
* Item Fat Content
* Item Type
* Sales
* Outlet Size and Establishment Year

---

## 🔍 Key Steps Performed

1. Data Cleaning

   * Handled missing values
   * Corrected inconsistent labels

2. Data Transformation

   * Grouped data using `groupby()`
   * Reshaped data using `unstack()`

3. Exploratory Data Analysis (EDA)

   * Compared sales across outlet tiers
   * Analyzed item category performance

4. Data Visualization

   * Bar charts for comparison
   * Pie charts for distribution

---

## 📊 Sample Analysis

* Sales comparison between **Low Fat** and **Regular** items
* Outlet tier-wise performance analysis
* Contribution of different product categories

---

## 📈 Key Insights

* Certain outlet tiers generate higher sales consistently
* Regular fat products tend to contribute more to revenue
* Sales distribution varies significantly across outlet types

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/your-username/blinkit-analysis.git
```

2. Navigate to project folder

```bash
cd blinkit-analysis
```

3. Install dependencies

```bash
pip install -r lib.txt
```

4. Run the script

```bash
python analysis.py
```
